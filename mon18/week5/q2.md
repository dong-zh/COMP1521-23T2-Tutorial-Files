# Week 5 Tutorial Question 2

## How do you tell an integer constant in C is in decimal, hexadecimal, octal, or binary?

```c
int x = 42; // decimal
int y = 0x42; // hex
int z = 08 // octal
int a = 0b1010 1000 // binary
```
